## 2. 结构约定

本次模型统一采用：

- Python 内部字段：`snake_case`
- 对外响应别名：`camelCase`

实现方式：

- 通过 `CamelModel`
- 使用 `alias_generator`

这样做的目的：

1. 保持 Python 代码风格自然。
2. 对齐前端已经定义好的 TypeScript 类型。
3. 减少后续接口实现时的字段转换成本。
