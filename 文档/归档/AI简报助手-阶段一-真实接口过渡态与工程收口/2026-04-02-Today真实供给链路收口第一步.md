## 3. 当前 Today 页数据来源判断

### 已进入真实接口过渡态

1. 用户兴趣
   - 优先读 `users.interests`
   - 无真实兴趣时才回退到虚拟兴趣态

2. 值得知道的
   - 来自 `hot_topics`

3. 值得行动的
   - 来自 `opportunities`

4. 因你关注而推荐
   - 基于真实 `hot_topics / opportunities` 与兴趣做匹配

### 仍然属于过渡态

1. 推荐排序逻辑
   - 当前是最小规则匹配
   - 不是正式推荐策略

2. 内容加工层
   - 当前没有统一内容清洗 / 去重 / 打分 / 编排层

3. 内容整合层
   - 当前只聚合热点与机会
   - 还没有统一文章层、briefing 层、内容详情层

4. 虚拟态仍未完全退场
   - 当 `users.interests` 为空时，Today 仍会回退到共享虚拟兴趣状态
