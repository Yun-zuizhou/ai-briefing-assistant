## 一、原型概述

### 1.1 设计目标

| 目标 | 说明 |
|------|------|
| 验证信息架构 | 确认页面结构和导航逻辑 |
| 测试用户流程 | 验证核心任务流程是否顺畅 |
| 沟通设计方案 | 与开发团队沟通设计意图 |
| 快速迭代 | 低保真便于快速修改 |

### 1.2 原型范围

| 页面 | 类型 | 优先级 |

| 输入区 | 输入框 + 快捷标签 | 用户输入 |
| 底部栏 | Tab导航 | 4个Tab入口 |
