## 一、设计系统概述

### 1.1 设计风格

本应用采用**报纸风格（Newspaper Style）**设计语言，融合传统报刊排版美学与现代移动端交互体验。

**核心设计理念：**
- 复古报刊质感：双线边框、衬线字体、纸张纹理
- 信息层级清晰：标题、摘要、正文层次分明
- 黑白为主色调：墨色（ink）与纸色（paper）的对比
- 点缀色彩强调：红色（accent-red）用于重点、蓝色（accent-blue）用于链接、金色（accent-gold）用于成就

### 1.2 设计令牌（Design Tokens）

#### 颜色系统

| 令牌名称 | 色值 | 用途 |
|---------|------|------|
| `--paper-50` | #FAFAF8 | 最浅纸色，背景色 |
| `--paper-100` | #F5F5F0 | 主背景色 |
| `--paper-200` | #EAEAE5 | 分隔线、边框 |
| `--paper-300` | #D4D4C8 | 次要边框 |
| `--paper-400` | #B8B8A8 | 辅助文字 |
| `--ink-700` | #3A3A3A | 正文文字 |
| `--ink-800` | #2C2C2C | 标题文字 |
| `--ink-900` | #1A1A1A | 主边框、强调文字 |
| `--accent-red` | #C41E3A | 重点强调、热点标识 |
| `--accent-blue` | #1E3A5F | 链接、次要强调 |
| `--accent-gold` | #B8860B | 成就、连续打卡 |
| `--success` | #2D5A27 | 成功状态 |
| `--warning` | #B8860B | 警告状态 |
| `--error` | #8B2500 | 错误状态 |

#### 字体系统

| 令牌名称 | 字体 | 用途 |
|---------|------|------|
| `--font-display` | Playfair Display, Noto Serif SC | 大标题、报头 |
| `--font-body` | Source Serif 4, Noto Serif SC | 正文、描述 |
| `--font-serif-cn` | Noto Serif SC, Source Serif 4 | 中文衬线 |

#### 字号系统

| 令牌名称 | 大小 | 用途 |
|---------|------|------|
| `xs` | 10px | 标签、辅助信息 |
| `sm` | 11px | 日期、来源 |
| `base` | 13px | 正文 |
| `md` | 14px | 卡片标题 |
| `lg` | 16px | 小标题 |
| `xl` | 18px | 中标题 |
| `2xl` | 24px | 页面标题 |
| `3xl` | 28px | 大标题 |
| `4xl` | 32px | 报头标题 |

#### 间距系统

| 令牌名称 | 大小 | 用途 |
|---------|------|------|
| `xs` | 4px | 紧凑间距 |
| `sm` | 8px | 元素内间距 |
| `md` | 12px | 卡片内间距 |
| `lg` | 16px | 区块间距 |
| `xl` | 20px | 大区块间距 |
| `2xl` | 24px | 页面边距 |

```
│  │                   mobile-device                          ││

│  │  ┌─────────────────────────────────────────────────────┐││
│  │  │                 mobile-screen                        │││
│  │  │  (100%宽高, 圆角36px, paper-100背景, 纸张纹理)       │││
│  │  │  ┌───────────────────────────────────────────────┐  │││
│  │  │  │              页面内容区域                       │  │││
│  │  │  └───────────────────────────────────────────────┘  │││

│  └─────────────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────────────┘
```

### 2.2 设备模拟器样式

**mobile-device（设备外壳）**
- 尺寸：375px × 812px
- 背景：深色渐变 `linear-gradient(145deg, #1A1A1A, #2C2C2C)`
- 圆角：44px
- 内边距：12px
- 阴影：多层阴影 + 红色光晕

**mobile-screen（屏幕区域）**
- 背景：`--paper-100` (#F5F5F0)
- 纹理：SVG噪点纹理（opacity: 0.03）
- 圆角：36px
- 溢出：hidden

## 十、设计规范总结

### 10.1 核心设计原则

1. **报纸美学**：双线边框、衬线字体、纸张纹理营造报刊质感
2. **信息层级**：清晰的标题-摘要-正文结构
3. **色彩克制**：黑白为主，点缀色仅用于强调
4. **交互反馈**：悬停、点击状态明确
5. **一致性**：统一的组件样式和间距系统
