# 首页、行动页、成长页对象消费边界判断

- 文档状态：进行中
- 建立时间：2026-04-05
- 所属阶段：阶段十 - 主闭环页面正式读侧整合与对象消费收口
- 文档职责：承接首页、行动页、成长页对已成立正式对象的消费差距判断

---

## 1. 当前要回答的问题

阶段十当前不先新增对象，而是先回答：

1. 首页是否真正消费了 `briefings`
2. 行动页是否真正消费了 `actions/overview -> following_items`
3. 成长页是否真正消费了 `growth-overview -> recent_history_items / user_profiles`

---

## 2. 当前页面消费现实

### 2.1 首页

当前判断：

- **基本已经消费到正式对象**

依据：

1. 后端 `GET /api/v1/dashboard/today` 已接 briefing
2. 前端 `TodayPage.tsx` 直接走 `apiService.getTodayPageData()`
3. 这条链路已经从对象层一路接到页面

结论：

首页当前不是阶段十最大的缺口。

### 2.2 行动页

当前判断：

- **已完成第一轮正式读侧切换**

依据：

1. 后端已有 `GET /api/v1/actions/overview`
2. 前端 `ActionsPage.tsx` 已改为直接调用：
   - `getActionsOverview()`
3. 页面当前已直接消费：
   - `todayTodos`
   - `savedForLater`
   - `followingItems`
   - `reminderSummary`
4. 为保持现有页面结构，原“待办 / 收藏待处理 / 打卡节律”都已改用正式对象字段就地映射
5. 同时补上了原本未真正落到页面上的：
   - `followingItems`
6. 当前仍保留原页面的本地筛选与勾选交互；其中“未来 / 已完成”视图，现阶段按 `actions/overview` 已返回的待办集合做本地视图承接，不再回退到旧接口拼装

结论：

行动页当前已开始真正消费正式的行动页读侧对象。

### 2.3 成长页

当前判断：

- **已完成第一轮正式读侧切换**

依据：

1. 后端已有 `GET /api/v1/preferences/growth-overview`
2. 前端 `GrowthPage.tsx` 已改为直接调用：
   - `getGrowthOverview()`
3. 页面当前已直接消费：
   - `growth-overview`
   - `recentHistoryItems`
   - `persona`
   - `reports`
4. 为保持现有页面结构，原“成长摘要 / 关键词 / 历史回顾 / 报告入口”均已改用正式对象字段就地映射
5. 当前仍未补新对象字段；原页面里“活跃关注项 / 历史总量”这类未在 `growth-overview` 中直接提供的字段，暂以正式对象中的关键词与历史预览口径承接

结论：

成长页当前已不再是阶段十最大的页面层缺口。

---

## 3. 当前结果级结论

阶段十当前的页面消费差距，应收口为：

1. 首页已经基本站到正式读侧上
2. 成长页已完成第一轮切到 `growth-overview`
3. 行动页已完成第一轮切到 `actions/overview`
4. `ActionsPage` 当前已进一步补齐正式分组读侧：
   - `todayTodos`
   - `futureTodos`
   - `completedTodos`
5. 首页、成长页、行动页这三条主闭环页面都已开始站到正式读侧上

---

## 4. 当前仍剩的页面级尾差

当前复核后，剩余差口不再是“页面还没切正式读侧”，而主要是以下尾差：

1. 首页 `TodayPage`
   - 已经走 `dashboard/today`
   - 当前剩余更多是内容组织与交互细节尾差，不是正式读侧主链缺口
2. 行动页 `ActionsPage`
   - 已切到 `actions/overview`
   - 当前“未来 / 已完成”筛选已经切到正式分组读侧
   - 这条尾差当前已完成收口
3. 成长页 `GrowthPage`
   - 已切到 `growth-overview`
   - 但原页面中的“活跃关注项 / 历史总量”仍是用正式对象现有字段做映射承接
   - 当前判断这类尾差继续后置，不再进入本轮实现

当前判断：

- 这些都属于第一轮切换后的尾差
- 不再构成阶段十前半段那种“页面还在手工拼旧接口”的主缺口
- 因此阶段十当前已经具备向子纲上推结果级结论的条件
