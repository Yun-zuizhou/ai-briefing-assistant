# 画像文案模板

---

## 一、画像主模板

### 模板：标准画像

```
在{year}年的时代图景中，你是一位关注{interests}的{role}。

{stance_description}

{growth_description}

你的时代关键词：{keywords}
```

**变量说明**：

| 变量 | 类型 | 说明 |
|------|------|------|
| year | string | 年份 |
| interests | string | 关注领域，用"、"连接 |
| role | string | 角色定位（思考者/探索者/行动派） |
| stance_description | string | 立场描述段落 |
| growth_description | string | 成长描述段落 |
| keywords | string | 关键词，用"、"连接 |

---

## 二、立场描述模板

### 模板：单一立场

```
当{topic}成为焦点时，你选择了"{stance}"的立场。
```

**示例**：
```
当AI监管成为焦点时，你选择了"理性看待"的立场。
```

### 模板：立场对比

```
当{topic1}引发讨论时，你选择了"{stance1}"；当{topic2}成为热点，你看到了"{stance2}"。
```

**示例**：
```
当AI监管引发讨论时，你选择了"理性看待"；当斜杠青年成为热点，你看到了"多元可能"。
```

### 模板：立场倾向总结

```
在立场选择上，你倾向于"{stance_tendency}"，{stance_explanation}。
```

**示例**：
```
在立场选择上，你倾向于"理性看待"，既看到机会也关注风险。
```

---

## 三、成长描述模板

### 模板：观点演变

```
从{old_time}的"{old_stance}"到{new_time}的"{new_stance}"，你对{topic}的思考在深入。
```

**示例**：
```
从年初的"观望"到年末的"支持"，你对AI监管的思考在深入。
```

### 模板：行动成长

```
你开始将想法付诸实践，加入了{plan_count}个成长计划，完成了{completed_count}个。
```

**示例**：
```
你开始将想法付诸实践，加入了12个成长计划，完成了8个。
```

### 模板：综合成长

```
{time_period}，你的{growth_area}在成长。{growth_detail}
```

**示例**：
```
这一年，你的思考在成长。从观望到行动，从犹豫到坚定。
```

## 四、角色定位模板

### 角色类型

| 角色 | 条件 | 描述 |
|------|------|------|
| 思考者 | 观点数>10，行动较少 | 善于思考，观点深入 |
| 探索者 | 关注领域>5 | 兴趣广泛，探索多元 |
| 行动派 | 计划完成率>70% | 行动力强，说到做到 |
| 记录者 | 活跃天数>100 | 持续记录，见证成长 |

### 模板：角色描述

```
你是一位{role}，{role_description}。
```

**示例**：

| 角色 | 描述 |
|------|------|
| 思考者 | 善于在变化中思考，用观点回应时代 |
| 探索者 | 对世界保持好奇，在探索中发现可能 |
| 行动派 | 将想法付诸实践，用行动抓住机会 |
| 记录者 | 持续记录成长轨迹，见证自己的变化 |

## 五、关键词模板

### 模板：三关键词

```
{keyword1} · {keyword2} · {keyword3}
```

**示例**：
```
独立思考 · 持续成长 · 拥抱变化
```

### 关键词库

| 类别 | 关键词 |
|------|--------|
| 思考类 | 独立思考、深度思考、理性分析 |
| 行动类 | 行动派、说到做到、持续行动 |
| 成长类 | 持续成长、不断进步、自我突破 |
| 探索类 | 勇于探索、保持好奇、多元尝试 |
| 态度类 | 拥抱变化、开放心态、积极乐观 |

## 六、特殊场景模板

### 模板：冷启动用户

```
你刚开始探索时代与自我的旅程。

每一次互动都在帮你更接近真实的自己。
继续前行吧，你的时代坐标正在形成中。
```

### 模板：数据不足

```
你已经在{app_usage_time}开始记录自己的时代印记。

继续关注热点、发表观点、抓住机会，
我会帮你描绘出更完整的时代坐标。
```

### 模板：高活跃用户

```
在{year}年，你与时代紧密相连。

{stats_summary}

你的每一次思考、每一个行动，都在书写你的时代故事。
```

## 七、完整示例

### 示例1：标准画像

```
在2024年的时代图景中，你是一位关注「AI变革」与「青年成长」议题的思考者。

当AI监管成为全球焦点时，你选择了"理性看待"的立场，认为"监管是必要的，但需要平衡创新空间"；当斜杠青年引发讨论时，你看到了"多元可能"，认为"多元发展更有安全感"。

从年初的"观望"到年末的"行动"，你的思考在成长。你开始将想法付诸实践，加入了3个成长计划，完成了2个。

你的时代关键词：独立思考、持续成长、拥抱变化
```

### 示例2：行动派画像

```
在2024年的时代图景中，你是一位关注「职业发展」与「技能提升」的行动派。

你加入了15个成长计划，完成了12个。你投递了8份简历，参加了5次面试，获得了2个机会。你用行动证明：想到，就去做。

你的行动风格是：快速决策、持续跟进、不轻言放弃。

你的时代关键词：行动派、说到做到、持续进步
```

### 示例3：探索者画像

```
在2024年的时代图景中，你是一位关注「AI技术」「职业发展」「学习方法」等多个领域的探索者。

你对世界保持好奇，在7个不同领域留下了思考的足迹。你善于发现机会，在变化中寻找可能。

你的探索轨迹：从AI技术到职业规划，从学习方法到个人成长。

你的时代关键词：勇于探索、保持好奇、多元尝试
```
