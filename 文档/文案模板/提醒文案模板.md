# 提醒文案模板

## 一、行动提醒模板

### 模板：轻度提醒（3天）

```
{plan_name}已经在你的计划里待了3天啦～今天要不要开始准备{action_item}？
```

**变量说明**：

| 变量 | 说明 |
|------|------|
| plan_name | 计划名称 |
| action_item | 行动项（简历/作品集/报名表） |

**示例**：
```
远程内容运营兼职已经在你的计划里待了3天啦～今天要不要开始准备简历？
```

### 模板：中度提醒（7天）

```
这个机会已经等了你一周了。{motivation}，要不要今天迈出第一步？
```

**motivation选项**：

| 选项 | 适用场景 |
|------|----------|
| 时间在走，机会也在走 | 通用 |
| 每一天都是新的开始 | 鼓励型 |
| 行动比完美更重要 | 完美主义者 |

**示例**：
```
这个机会已经等了你一周了。时间在走，机会也在走，要不要今天迈出第一步？
```

### 模板：重度提醒（14天）

```
{plan_name}已经在计划里待了两周。{gentle_push}
```

**gentle_push选项**：

| 选项 | 语气 |
|------|------|
| 现在开始，还不晚。 | 鼓励 |
| 也许这个机会不太适合现在的你？没关系，看看别的也可以。 | 释怀 |
| 有什么困难吗？我可以帮你想想办法。 | 关心 |

**示例**：
```
远程内容运营兼职已经在计划里待了两周。现在开始，还不晚。
```

## 七、文案风格检查清单

### ✅ 正确示例

| 类型 | 文案 | 说明 |
|------|------|------|
| 行动提醒 | 今天要不要开始准备简历？ | 鼓励、引导 |
| 截止提醒 | 还有3天截止，记得预留时间 | 紧迫但不焦虑 |
| 放弃建议 | 放弃也是一种努力 | 理解、释怀 |
| 成功庆祝 | 太棒了！你的努力有了回报 | 肯定、庆祝 |

### ❌ 错误示例

| 类型 | 文案 | 问题 |
|------|------|------|
| 行动提醒 | 你已经拖了3天了，快点行动！ | 指责、催促 |
| 截止提醒 | 再不投递就来不及了！ | 制造焦虑 |
| 放弃建议 | 你太懒了，放弃吧 | 评判、否定 |
| 成功庆祝 | 你终于成功了！ | 暗示之前不成功 |
