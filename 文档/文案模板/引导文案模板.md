## 五、提示文案模板

### 模板：加载中

```
⏳ 正在生成你的时代坐标...

每一次思考，都在塑造你的画像。
```

### 模板：数据更新

```
🔄 数据已更新

发现了{new_count}个新的{content_type}！
```

**content_type选项**：

| 选项 | 场景 |
|------|------|
| 热点 | 热点更新 |
| 机会 | 机会更新 |
| 内容 | 通用 |

### 模板：操作成功

```
✅ {action}成功！

{next_step}
```

**action和next_step组合**：

| action | next_step |
|--------|-----------|
| 加入计划 | 我会提醒你截止日期，加油！ |
| 发表观点 | 你的观点已记录，继续思考吧！ |
| 完成计划 | 太棒了！你的努力有了回报！ |

### 模板：操作失败

```
❌ {action}失败

{reason}

{retry_suggestion}
```

**reason和retry_suggestion组合**：

| reason | retry_suggestion |
|--------|------------------|
| 网络连接失败 | 请检查网络后重试 |
| 服务器繁忙 | 请稍后再试 |
| 数据加载失败 | 下拉刷新试试 |

## 六、帮助文案模板

### 模板：什么是时代坐标？

```
📍 什么是时代坐标？

时代坐标是你的成长画像，包含：

• 兴趣领域 - 你关注的话题
• 立场倾向 - 你对议题的态度
• 行动风格 - 你的行动力
• 成长轨迹 - 你的变化趋势
• 时代关键词 - 你的标签

持续使用APP，你的时代坐标会越来越清晰。
```

### 模板：什么是时代计划？

```
📋 什么是时代计划？

时代计划帮你追踪机会的执行进度：

• 加入计划 - 把机会加入追踪列表
• 状态更新 - 记录你的行动进展
• 智能提醒 - 提醒你截止日期和行动
• 完成记录 - 记录你的成功时刻

我会陪伴你完成每一个计划。
```

### 模板：如何获得更好的画像？

```
💡 如何获得更好的画像？

1. 关注热点 - 浏览和点击感兴趣的热点
2. 发表观点 - 对热点发表你的看法
3. 加入计划 - 把机会加入追踪列表
4. 完成行动 - 按时完成你的计划

你的每一次互动，都会让画像更准确。
```

## 八、文案风格检查清单

### ✅ 正确示例

| 类型 | 文案 | 说明 |
|------|------|------|
| 欢迎引导 | 让我们一起，记录你在时代中的每一次思考 | 温暖、有愿景 |
| 功能介绍 | 我会帮你追踪截止日期，陪伴你的成长 | 清晰、有价值 |
| 空状态 | 继续前行吧，你的时代坐标正在形成中 | 鼓励、有希望 |
| 帮助说明 | 你的每一次互动，都会让画像更准确 | 清晰、有指导 |

### ❌ 错误示例

| 类型 | 文案 | 问题 |
|------|------|------|
| 欢迎引导 | 这是一个记录APP | 无情感、无吸引力 |
| 功能介绍 | 你可以点击这里查看热点 | 无价值感 |
| 空状态 | 你还没有数据 | 冷漠、无引导 |
| 帮助说明 | 画像就是你的数据统计 | 无情感、无指导 |
