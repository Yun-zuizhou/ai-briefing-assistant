# 文案模板说明文档

## 概述

本文件夹包含"时代与我"APP的所有文案模板。这些模板用于生成各场景下的标准化文案，确保文案风格统一、质量稳定。

## 文案风格指南

### 核心原则

| 原则 | 说明 | 示例 |
|------|------|------|
| **第二人称** | 使用"你"而非"用户" | "你是一位关注..." |
| **客观记录者** | 避免主观评价 | "数据显示你关注..." |
| **成长导向** | 强调变化和成长 | "你的思考在成长" |
| **温暖陪伴** | 情感化表达 | "放弃也是一种努力" |
| **时代感** | 结合时代背景 | "在2024年的时代图景中..." |

### 语气规范

| 场景 | 语气 | 示例 |
|------|------|------|
| 画像描述 | 客观、温暖 | "你是一位关注AI变革的思考者" |
| 行动提醒 | 鼓励、引导 | "今天要不要开始准备简历？" |
| 截止提醒 | 紧迫但不焦虑 | "还有3天截止，记得预留时间" |
| 放弃建议 | 理解、释怀 | "放弃也是一种努力" |
| 成功庆祝 | 肯定、庆祝 | "太棒了！你的努力有了回报" |

### 禁用词汇

| 类型 | 禁用词 | 替代词 |
|------|--------|--------|
| 负面评价 | 失败、懒惰、拖延 | 未完成、待行动 |
| 命令语气 | 必须、一定要 | 建议、可以试试 |
| 焦虑制造 | 再不...就...、最后机会 | 记得、别忘了 |
| 主观评判 | 你应该、你错了 | 可以考虑、也许 |

## 使用方式

### 模板调用

```python
class CopywritingService:
    """文案服务"""
    
    def get_profile_copy(self, template_name: str, variables: dict) -> str:
        """获取画像文案"""
        template = self.load_template(f"画像文案模板.md", template_name)
        return self.render(template, variables)
    
    def get_reminder_copy(self, reminder_type: str, variables: dict) -> str:
        """获取提醒文案"""
        template = self.load_template(f"提醒文案模板.md", reminder_type)
        return self.render(template, variables)
    
    def get_guide_copy(self, guide_name: str, variables: dict) -> str:
        """获取引导文案"""
        template = self.load_template(f"引导文案模板.md", guide_name)
        return self.render(template, variables)
```

### 变量替换

模板中的变量使用 `{变量名}` 格式：

```
模板：在{year}年的时代图景中，你是一位关注{interests}的{role}。

调用：render(template, {"year": "2024", "interests": "AI变革", "role": "思考者"})

输出：在2024年的时代图景中，你是一位关注AI变革的思考者。
```
