# 项目核心总纲

- 项目名称：AI简报助手（时代与我）
- 文档定位：项目级唯一核心口径
- 当前版本：v3.1
- 更新日期：2026-04-22
- 本次更新：高频一键命令入口与文档功能区分层导航已同步

## 1. 项目一句话定义

AI简报助手（时代与我）是一个围绕"今日信息 -> 行动转化 -> 自我记录 -> 周期回顾"运行的个人信息助手系统。

## 2. 当前稳定核心

当前已经进入稳定主线的一级能力：

1. 首页正式读侧：`dashboard/today`
2. 行动页正式读侧：`actions/overview`
3. 成长页正式读侧：`preferences/growth-overview`
4. 统一内容详情正式读侧：`content/by-ref`
5. Chat 正式消息与会话链路
6. `reports / briefings / user_profiles / opportunity_follows` 第一轮正式承接位
7. 历史报告列表与按 `report_id` 读取的周期回顾链路

## 3. 当前必须保持的原则

1. 主线稳定后，不回退到旧接口拼装或无序扩面
2. 正式后端链路不再写回 demo mock 文件
3. `npm.cmd run check` 是唯一准入门槛
4. Windows 标准运行入口只保留 `setup / dev / check`
5. 当前工作区根目录就是唯一正式仓库根
