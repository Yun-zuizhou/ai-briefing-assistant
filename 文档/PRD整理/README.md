# PRD整理目录

- 01_用户故事（200-800字）
  - 形式：用户故事 + 验收标准 + 简要说明
  - 适用：小迭代或内部工具

- 02_单一功能模块（1500-5000字）
  - 示例：登录注册、搜索筛选、个人中心改版
  - 内容：流程图、规则、异常处理

- 03_完整项目_中型产品（8000-25000字）
  - 内容：业务背景、用户角色、全功能列表、详细用例、非功能需求、数据埋点
  - 可分多个文档或章节

- 04_大型平台_系统级PRD（30000字以上）
  - 内容：多子系统、复杂权限、合规要求、接口协议
  - 建议：主文档 + 子模块文档 + BRD/技术设计文档协同
