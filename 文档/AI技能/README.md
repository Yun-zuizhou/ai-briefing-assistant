# AI Skills 说明文档

---

## 概述

本文件夹包含"时代与我"APP的所有AI技能设计文档。这些Skills是产品的核心竞争力，负责生成用户画像、归纳观点、生成年度报告等关键功能。

---

## Skills 列表

| Skill | 文件 | 用途 | 触发场景 |
|-------|------|------|----------|
| **画像生成** | [画像生成Skill.md](./画像生成Skill.md) | 生成"时代感"用户画像 | 用户查看画像、定期更新 |
| **观点小结** | [观点小结Skill.md](./观点小结Skill.md) | 归纳用户观点轨迹 | 用户查看观点记录页 |
| **年度报告** | [年度报告Skill.md](./年度报告Skill.md) | 生成年度时代印记 | 年末生成、用户主动查看 |
| **智能提醒** | [智能提醒Skill.md](./智能提醒Skill.md) | 生成情感化提醒文案 | 计划提醒、截止提醒 |

## 设计原则

### 1. 文案风格

| 原则 | 说明 | 示例 |
|------|------|------|
| **第二人称** | 使用"你"而非"用户" | "你是一位关注..." |
| **客观记录者** | 避免主观评价，用数据说话 | "数据显示你关注..." |
| **成长导向** | 强调变化和成长 | "你的思考在成长" |
| **时代感** | 结合时代背景描述 | "在2024年的时代图景中..." |
| **温暖陪伴** | 情感化表达，有温度 | "放弃也是一种努力" |

### 2. 分析维度

| 维度 | 说明 | 数据来源 |
|------|------|----------|
| **兴趣领域** | 用户关注的话题 | 热点点击、观点输出 |
| **立场倾向** | 用户对议题的态度 | 观点立场选择 |
| **行动风格** | 用户的行动力 | 计划完成率、行动速度 |
| **成长轨迹** | 用户的变化趋势 | 观点演变、能力提升 |
| **时代关键词** | 用户的标签 | 综合分析 |

### 3. 输出格式

| Skill | 输出格式 | 字数限制 |
|-------|----------|----------|
| 画像生成 | 文字描述 + 关键词 | 200-300字 |
| 观点小结 | 段落描述 | 100-150字 |
| 年度报告 | 多段落 + 数据 | 500-800字 |
| 智能提醒 | 单句/短段落 | 20-50字 |
