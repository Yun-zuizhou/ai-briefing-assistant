## 一、品牌概述

### 1.1 品牌定位

**AI简报助手** —— 每天一份简报，帮你收集世界，记录自己，看见成长

### 1.2 品牌关键词

| 关键词 | 说明 |
|--------|------|
| **智能** | AI驱动，自动化信息处理 |
| **简洁** | 简报为核心，信息精炼 |
| **成长** | 记录轨迹，见证进步 |
| **温暖** | 个人叙事，情感连接 |

### 1.3 设计原则

| 原则 | 说明 |
|------|------|
| **内容优先** | 简报内容是核心，UI服务于内容 |
| **渐进式引导** | 从简单到复杂，逐步引导用户 |
| **即时反馈** | 每个行为都有反馈 |
| **一致性** | 统一的视觉语言和交互模式 |

## 七、动画系统

### 7.1 时长

| 名称 | 值 | 用途 |
|------|------|------|
| **fast** | 150ms | 微交互 |
| **normal** | 200ms | 标准过渡 |
| **slow** | 300ms | 复杂动画 |
| **slower** | 500ms | 页面过渡 |

### 7.2 缓动函数

| 名称 | 值 | 用途 |
|------|------|------|
| **ease** | `cubic-bezier(0.4, 0, 0.2, 1)` | 默认 |
| **ease-in** | `cubic-bezier(0.4, 0, 1, 1)` | 进入 |
| **ease-out** | `cubic-bezier(0, 0, 0.2, 1)` | 退出 |
| **bounce** | `cubic-bezier(0.68, -0.55, 0.265, 1.55)` | 弹性 |

### 7.3 动画效果

```css
/* 淡入 */
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

/* 上滑进入 */
@keyframes slideUp {
  from { opacity: 0; transform: translateY(20px); }
  to { opacity: 1; transform: translateY(0); }
}

/* 缩放进入 */
@keyframes scaleIn {
  from { opacity: 0; transform: scale(0.95); }
  to { opacity: 1; transform: scale(1); }
}

/* 脉冲 */
@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.5; }
}
```

## 十二、无障碍规范

### 12.1 对比度

| 元素 | 最小对比度 |
|------|-----------|
| 正文文字 | 4.5:1 |
| 大标题 | 3:1 |
| 图标 | 3:1 |

### 12.2 焦点状态

- 所有可交互元素必须有可见的焦点状态
- 焦点环使用主色 + 2px 外轮廓
- Tab顺序与视觉顺序一致

### 12.3 触摸目标

- 最小触摸目标：44x44px
- 触摸目标间距：8px
