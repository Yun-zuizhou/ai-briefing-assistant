## 设计原则

### 核心理念

**Slogan**：*"记录你在时代中的每一次思考"*

```
时代（热点/趋势/事件）←→ 我（观点/行动/成长）
```

### 设计标准

| 维度 | 标准 |
|------|------|
| **情感基调** | 温暖、陪伴感、故事性 |
| **视觉风格** | 简洁、现代、有温度 |
| **交互原则** | 低门槛、高价值、持续陪伴 |
| **文案风格** | 第二人称、客观记录者语气、成长导向 |

### 用户使用流程

```
首页（对话式）
    ↓ 触发
┌───────────────────────────────────────┐
│ 时代热点Tab                           │
│ ├─ 热点列表 → 热点详情 → 观点输入     │
│ └─ 观点记录 → 时间线 + AI小结         │
├───────────────────────────────────────┤
│ 成长机遇Tab                           │
│ ├─ 机会卡片 → 机会详情 → 加入计划     │
│ └─ 计划卡组 → 状态追踪 → 智能提醒     │
├───────────────────────────────────────┤
│ 我的画像Tab                           │
│ ├─ 画像展示 → 文字描述 + 雷达图       │
│ └─ 个人设置 → 年度报告入口            │
└───────────────────────────────────────┘
```

## 一、首页设计

### 1.1 页面定位

**定位**：对话式交互入口，快速触达各功能

**核心价值**：
- 降低使用门槛（人人都会聊天）
- 统一入口（所有功能通过对话触达）
- 冷启动友好（新用户可直接问"你能帮我做什么？"）

### 1.2 页面结构

```
┌─────────────────────────────────────┐
│  🌍 时代与我                    ⚙️   │  ← 顶部导航栏
├─────────────────────────────────────┤
│                                     │
│  [AI] 早安！今天有几个时代热点       │  ← 对话区域
│       想和你聊聊：                   │
│                                     │
│       ┌─────────────────────┐       │
│       │ 🔥 AI监管新规出台    │       │  ← 嵌入式卡片
│       │ 你怎么看？          │ →     │
│       └─────────────────────┘       │
│                                     │
│  [你] 我对AI监管比较感兴趣          │
│                                     │
│  [AI] 好的！AI监管新规主要涉及...   │
│                                     │
├─────────────────────────────────────┤
│  ┌─────────────────────────────────┐│
│  │ 💬 问我任何事...                ││  ← 输入框
│  └─────────────────────────────────┘│
│  [热点] [机会] [画像] [更多]        │  ← 快捷标签
├─────────────────────────────────────┤
│ [首页] [时代热点] [成长机遇] [我的画像]│  ← 底部Tab
└─────────────────────────────────────┘
```

### 1.3 设计规范

#### 顶部导航栏

| 元素 | 规范 |
|------|------|
| 高度 | 56px |
| 背景 | 主色调渐变 |
| 标题 | "时代与我" + Logo图标 |
| 右侧 | 设置图标（齿轮） |

#### 对话区域

| 元素 | 规范 |
|------|------|
| 背景 | 浅灰/白色（支持深色模式） |
| AI气泡 | 左对齐，圆角16px，背景#F5F5F5 |
| 用户气泡 | 右对齐，圆角16px，背景主色调 |
| 字体 | 16px，行高1.5 |
| 间距 | 气泡间距12px |

#### 嵌入式卡片

| 元素 | 规范 |
|------|------|
| 圆角 | 12px |
| 阴影 | 0 2px 8px rgba(0,0,0,0.1) |
| 内边距 | 16px |
| 标题 | 16px，加粗 |
| 描述 | 14px，灰色 |
| 按钮 | 文字按钮，主色调 |

#### 输入框

| 元素 | 规范 |
|------|------|
| 高度 | 48px |
| 圆角 | 24px |
| 背景 | #F5F5F5 |
| 占位符 | "问我任何事..." |
| 快捷标签 | 胶囊按钮，可滑动 |

### 1.4 交互设计

#### 对话流程

```
用户进入首页
    ↓
AI主动问候 + 推送今日热点
    ↓
用户回复/点击卡片
    ↓
AI响应 + 引导下一步
    ↓
用户继续对话或跳转详情页
```

#### 快捷标签功能

| 标签 | 触发行为 |
|------|----------|
| 热点 | 快速输入"今天有什么热点？" |
| 机会 | 快速输入"帮我看看机会" |
| 画像 | 快速输入"我的时代坐标是什么？" |
| 更多 | 展开更多快捷指令 |

### 9.9 边缘情况组件

#### 9.9.1 EmptyState 空状态组件

**用途**：展示空状态页面

**属性**：

| 属性 | 类型 | 说明 |
|------|------|------|
| icon | string | 图标（emoji或图片URL） |
| title | string | 标题 |
| description | string | 描述文字 |
| actionText | string? | 操作按钮文字 |
| onAction | function? | 操作回调 |

**使用示例**：

```tsx
<EmptyState
  icon="📝"
  title="还没有观点记录"
  description="去看看今天的热点，留下你的思考吧"
  actionText="去浏览热点"
  onAction={() => navigateToHotTopics()}
/>
```

#### 9.9.2 ConfirmDialog 确认弹窗组件

**用途**：展示确认弹窗

**属性**：

| 属性 | 类型 | 说明 |
|------|------|------|
| icon | string | 图标 |
| title | string | 标题 |
| description | string | 描述 |
| cancelText | string | 取消按钮文字 |
| confirmText | string | 确认按钮文字 |
| confirmType | enum | 确认按钮类型（primary/danger） |
| onCancel | function | 取消回调 |
| onConfirm | function | 确认回调 |

**使用示例**：

```tsx
<ConfirmDialog
  icon="⚠️"
  title="确定要放弃这个计划吗？"
  description="放弃后可以从"机会"中重新发现"
  cancelText="取消"
  confirmText="确定放弃"
  confirmType="danger"
  onCancel={() => closeDialog()}
  onConfirm={() => abandonPlan()}
/>
```

#### 9.9.3 Toast 提示组件

**用途**：展示轻量级提示

**属性**：

| 属性 | 类型 | 说明 |
|------|------|------|
| message | string | 提示内容 |
| type | enum | 类型（success/error/warning） |
| duration | number | 显示时长（毫秒） |
| icon | string? | 自定义图标 |

**使用示例**：

```tsx
Toast.show({
  message: "观点已记录",
  type: "success",
  duration: 2000
});
```
