## 四、已确定的内容

> 说明：产品定位文档中明确的内容 + 本次对话确定的内容

### 4.1 产品定位（已确定）

- 产品名称：简报助手App
- 核心概念：时代（热点/趋势/事件）←→ 我（观点/行动/成长）
- Slogan：每天一份简报，帮你收集世界，记录自己，看见成长
- 目标用户：学术研究者、求职者、行业关注者、自我成长者

### 4.2 核心功能（已确定）

- 每日简报：信息追踪 + 待办提醒 + 个人叙事
- 简报互动：回复记录、收藏内容、创建待办
- 简报管理：历史查看、导出简报、分享简报
- 简报设置：推送时间、内容偏好、关注管理

### 4.3 推送机制（已确定）

- 晨间简报：8:00，昨夜收集 + 今日待办 + 昨日叙事
- 晚间简报：21:00，今日收集 + 明日待办 + 今日叙事
- 即时简报：重要事件发生时

### 4.4 设计原则（已确定）

- 简报驱动：所有功能围绕简报展开
- 被动触达：推送机制，用户无需主动
- 互动闭环：回复简报即可完成操作
- 对话式交互：所有输入通过对话完成（定义待澄清）

---

### 4.5 信息架构（本次确定 ✅）

#### 导航结构

| 按钮           | 页面内容             | 说明                                   |
| -------------- | -------------------- | -------------------------------------- |
| **对话** | AI对话界面           | 所有操作的入口，可查看历史对话         |
| **简报** | 今日简报（默认首页） | 热点讨论度 + 用户关注领域摘要          |
| **日志** | 个人叙事             | 用户记录的想法、成长轨迹               |
| **待办** | 待办管理 + 打卡      | 所有待办（今日/未来/已完成）+ 连续打卡 |
| **我的** | 收藏 + 设置          | 收藏内容、用户画像、偏好设置           |

#### 简报页面结构

```
简报页面（首页）
├── 热点话题区域（讨论度Top10）
│   ├── 只显示讨论度/热度变化
│   ├── 不直接显示摘要
│   └── 点击进入才有摘要和原文链接
├── 用户关注区域（主体）
│   ├── 区分不同领域和主题
│   ├── 领域切换：左右滑动或标签页
│   ├── 各领域的综合摘要
│   ├── 变化摘要（信息变化趋势）
│   └── 原文链接（可跳转）
└── 底部：对话输入框（全局入口）
```

**简报内容原则**：

- 主要呈现用户添加"关注"的内容
- 搜集文章、咨询、总结摘要、整理信息
- 不是推送所有"热点"，而是推送用户关注领域的相关内容
- 呈现各个领域热点信息变化趋势和话题热度

#### 交互方式

| 操作类型 | 交互方式               | 说明               |
| -------- | ---------------------- | ------------------ |
| 创建待办 | 对话优先，也可直接点击 | "明天提醒我投简历" |
| 添加关注 | 对话优先               | "我想关注AI前沿"   |
| 记录想法 | 对话优先               | "今天学到了..."    |
| 修改设置 | 对话优先               | "每天8点发简报"    |
| 完成待办 | 直接点击               | 点击复选框         |
| 收藏内容 | 直接点击               | 点击收藏按钮       |
| 查看原文 | 直接点击               | 点击链接           |
| 切换领域 | 直接点击               | 滑动或点击标签     |

#### 日志页面结构

```
日志页面
├── 今日叙事（AI生成 + 用户补充）
├── 成长轨迹（时间线）
└── 想法记录（列表）
```

#### 待办页面结构

```
待办页面
├── 打卡区域（顶部）
│   ├── 打卡按钮
│   └── 连续打卡天数
├── Tab切换
│   ├── 今日
│   ├── 未来
│   └── 已完成
└── 待办列表
```

#### 我的页面结构

```
我的页面
├── 用户信息
├── 用户画像（雷达图）
├── 收藏内容
├── 偏好设置
│   ├── 推送时间
│   ├── 关注管理
│   └── 通知设置
└── 其他（帮助、关于、退出）
```
