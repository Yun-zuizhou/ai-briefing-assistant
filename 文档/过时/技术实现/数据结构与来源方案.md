| 数据类型 | 来源方式 | 质量保证 |

| 热点资讯 | 爬虫抓取 + API筛选 | 内容摘要 + 原文链接 |

## 十一、技术实现

### 11.1 爬虫架构

```python
# 爬虫服务架构
class CrawlerService:
    """爬虫服务"""
    
    def crawl_hot_topics(self):
        """抓取热点事件"""
        # 1. 从多个平台抓取
        raw_data = []
        for platform in ['weibo', 'zhihu', 'baidu']:
            data = self.crawl_platform(platform)
            raw_data.extend(data)
        
        # 2. 去重
        unique_data = self.deduplicate(raw_data)
        
        # 3. 质量筛选
        filtered_data = self.filter_quality(unique_data)
        
        # 4. 生成摘要
        for item in filtered_data:
            item['summary'] = self.generate_summary(item)
        
        # 5. 入库
        self.save_to_db(filtered_data)
    
    def generate_summary(self, item):
        """生成内容摘要"""
        # 调用LLM生成摘要
        prompt = f"请用100字以内总结以下内容：{item['content']}"
        return self.llm_generate(prompt)
```

### 11.2 API接口

```python
# 热点API
@router.get("/hot-topics")
def get_hot_topics(
    category: str = None,
    limit: int = 20,
    db: Session = Depends(get_db)
):
    """获取热点列表"""
    query = db.query(HotTopic).filter(HotTopic.expires_at > datetime.now())
    
    if category:
        query = query.filter(HotTopic.categories.contains(category))
    
    topics = query.order_by(HotTopic.published_at.desc()).limit(limit).all()
    
    return [{
        "id": t.id,
        "title": t.title,
        "summary": t.summary,
        "source": t.source,
        "source_url": t.source_url,
        "categories": json.loads(t.categories),
        "guide_questions": json.loads(t.guide_questions),
        "hot_comments": json.loads(t.hot_comments),
        "public_opinions": json.loads(t.public_opinions)
    } for t in topics]
```

## 十二、注意事项

### 12.1 法律合规

| 事项 | 措施 |
|------|------|
| 版权问题 | 生成摘要，保留原文链接 |
| 数据来源 | 明确标注来源平台 |
| 用户隐私 | 不收集敏感信息 |
| 爬虫频率 | 控制请求频率，避免对目标网站造成压力 |

### 12.2 数据质量

| 指标 | 标准 |
|------|------|
| 内容完整性 | 标题、摘要、链接齐全 |
| 时效性 | 热点事件24小时内更新 |
| 准确性 | 摘要准确反映原文内容 |
