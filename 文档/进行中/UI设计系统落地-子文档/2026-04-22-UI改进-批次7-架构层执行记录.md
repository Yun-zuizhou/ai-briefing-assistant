```tsx
interface AppState {
  // ... existing state
  theme: 'light' | 'dark' | 'system';
}

interface AppContextType {
  // ... existing methods
  setTheme: (theme: 'light' | 'dark' | 'system') => void;
}

// 初始化时读取localStorage或系统偏好
const getInitialTheme = (): 'light' | 'dark' | 'system' => {
  const saved = localStorage.getItem('theme');
  if (saved === 'light' || saved === 'dark' || saved === 'system') return saved;
  return 'system';
};

// 监听系统主题变化
useEffect(() => {
  const mediaQuery = window.matchMedia('(prefers-color-scheme: dark)');
  const handleChange = () => {
    if (state.theme === 'system') {
      // 触发重新渲染以应用新主题
      setState(prev => ({ ...prev }));
    }
  };
  mediaQuery.addEventListener('change', handleChange);
  return () => mediaQuery.removeEventListener('change', handleChange);
}, [state.theme]);
```

## 3. 响应式策略

| 断点 | 范围 | 策略 |
|------|------|------|
| 默认 | < 360px | 保持当前移动端布局，允许横向滚动 |
| sm | 360px - 430px | 当前目标设备，完美适配 |
| md | 430px - 768px | 保持430px宽度，两侧留白 |
| lg | > 768px | 居中显示，添加阴影，优化大屏体验 |

### 3.1 大屏优化内容

1. **居中容器**：#root保持max-width:430px，margin:0 auto
2. **阴影层级**：添加box-shadow提升层级感
3. **背景色**：body使用更深的颜色，与#root形成对比
4. **鼠标优化**：hover状态更明显，适合鼠标交互

## 4. 暗色模式策略

### 4.1 颜色映射

| 浅色模式 | 暗色模式 | 用途 |
|---------|---------|------|
| #F5F2E8 (paper) | #1a1612 | 主背景 |
| #EDE9DC (paper-warm) | #2a2420 | 卡片背景 |
| #E5E0D0 (paper-dark) | #3a3428 | 阴影/边框 |
| #1A1612 (ink) | #e8e0d4 | 主文字 |
| #8B7D66→#6B5F4A (ink-muted) | #a09888 | 次要文字 |
| #C4A86B (accent) | #c4a86b | 强调色（保持） |

### 4.2 实现方式

1. **CSS媒体查询**：`@media (prefers-color-scheme: dark)`
2. **data属性覆盖**：`[data-theme="dark"]`
3. **系统监听**：`matchMedia('(prefers-color-scheme: dark)')`
