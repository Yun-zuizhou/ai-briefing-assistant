      </div>
      {meta && (

  icon: LucideIcon;
  onClick: () => void;
  ariaLabel: string;
  variant?: 'default' | 'primary';
  size?: number;
}

  return (
    <button
      onClick={onClick}
      aria-label={ariaLabel}
      className={`inline-flex items-center justify-center p-2 transition-colors ${
        variant === 'primary'
          ? 'bg-accent text-paper hover:bg-ink'
          : 'bg-paper text-ink-muted hover:text-ink'

}

## 3. 圆角/阴影统一决策

| 元素 | 圆角 | 阴影 | 决策依据 |
|------|------|------|---------|
| 卡片（Card） | 0 | `2px 2px 0 var(--paper-dark)` | 复古报纸风格 |
| 按钮（Button） | 0 | `2px 2px 0 var(--paper-dark)` | 与卡片保持一致 |
| 标签（Tag） | 0 | 无 | 降低视觉重量 |
| 输入框（Input） | 4px | 无 | 轻微圆角提升输入体验 |
| 模态框（Modal） | 0 | `0 4px 20px rgba(26, 22, 18, 0.08)` | 保持复古但需层级感 |
| 内容卡片（ContentCard） | 0 | `3px 3px 0 var(--paper-dark)` | 与domain-card一致 |
